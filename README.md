# 个人记账本 (Personal Account Book)

这是一个基于 Android 原生开发的个人记账 App。

## ✍️ 项目简介
我是来自（林涉外）的一名学生。本项目是在老师的悉心指导下，通过 5 天的学习与实战演练独立开发完成的。

该项目涵盖了 Android 开发的核心知识点，旨在帮助初学者掌握从 UI 布局、数据库存储到数据可视化的完整开发流程。

## ✨ 主要功能
*   **📋 账单管理**：支持每一笔收入与支出的添加、编辑和删除。
*   **📊 统计分析**：
    *   **饼图**：直观展示不同分类（如餐饮、购物、工资）的支出占比。
    *   **折线图**：展示最近 6 个月的收支趋势，方便进行财务分析。
*   **🔍 智能搜索与筛选**：可以按关键词搜索备注，或按月份筛选特定时间的账单。
*   **💾 数据持久化**：使用 Room 数据库，确保退出 App 后数据依然安全保存。

## 🛠️ 技术栈
*   **语言**：Kotlin
*   **架构**：MVVM (ViewModel + LiveData)
*   **数据库**：Room Persistence Library
*   ** UI 组件**：ViewBinding, Material Design, BottomSheetDialog
*   **图表库**：MPAndroidChart
